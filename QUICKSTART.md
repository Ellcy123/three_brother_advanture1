# 快速启动指南

## 🎯 5分钟快速运行

### 方法一：本地开发（最快）

```bash
# 1. 安装依赖（首次运行）
npm install
cd client && npm install && cd ..

# 2. 启动后端服务器
npm start

# 3. 新开一个终端，启动前端
cd client
npm start
```

浏览器会自动打开 `http://localhost:3000`

### 方法二：Railway一键部署

1. Fork这个仓库到你的GitHub
2. 访问 [railway.app](https://railway.app/)
3. 点击 "New Project" → "Deploy from GitHub repo"
4. 选择你fork的仓库
5. 等待部署完成（约2-3分钟）
6. 访问Railway提供的URL

## 🎮 如何开始游戏

### 1. 创建房间

- 打开游戏主页
- 点击"创建房间"
- 输入你的昵称
- 记住房间号（6位字符）

### 2. 邀请好友

- 将房间号分享给2位好友
- 好友点击"加入房间"
- 输入房间号和昵称
- 等待3人到齐

### 3. 开始冒险

- 3人到齐后游戏自动开始
- 注意查看谁的回合
- 轮到你时输入关键词（例如：`水潭+龟`）
- 观察效果，收集线索

### 4. 通关目标

- 找到4个字母：C、E、H、O
- 组成密码：ECHO
- 打开大门逃出密室！

## 🐛 遇到问题？

### 连接不上服务器

```bash
# 检查服务器是否启动
# 应该看到：服务器运行在端口 3001

# 检查端口是否被占用
# Windows
netstat -ano | findstr 3001

# Mac/Linux
lsof -i :3001
```

### 游戏卡住不动

- 刷新页面
- 检查网络连接
- 查看浏览器控制台是否有错误

### 断线重连

- 60秒内刷新页面会自动重连
- 超过60秒需要重新加入房间

## 🎯 游戏技巧

### 1. 团队协作

- 在外部聊天工具（微信/Discord）讨论策略
- 明确分工：谁负责探索、谁负责收集线索
- 注意避免重复触发相同组合

### 2. 生命值管理

- 小心触发负面效果（扣血组合）
- 保留一定生命值用于复活队友
- 部分组合可以回血，善加利用

### 3. 线索收集

- 记录哪些组合已经尝试过
- 某些组合会给出提示信息
- 系统提示的组合往往很重要

### 4. 时间管理

- 游戏没有时间限制，慢慢探索
- 但房间会在1小时无活动后清理
- 断线玩家有60秒重连时间

## 🔧 开发者选项

```javascript

  console.log(event, args);
});
```

```javascript
// 修改初始生命值

// 修改复活消耗
reviver.hp -= 2  // 改为其他数值
```
